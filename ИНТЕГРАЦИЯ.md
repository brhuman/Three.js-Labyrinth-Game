
### 3. Отладка интеграции

После загрузки страницы в консоли будут доступны функции:

```javascript
// Переключение архитектуры
window.gameDebug.toggleArchitecture(true);  // Новая архитектура
window.gameDebug.toggleArchitecture(false); // Старая архитектура

// Статистика интеграции
window.gameDebug.getIntegrationStats();

// Полная интеграция
window.gameDebug.fullIntegration();

// Откат интеграции
window.gameDebug.rollbackIntegration();

// Прямой доступ к компонентам
window.gameDebug.eventBus.emit('test_event', { data: 'test' });
window.gameDebug.stateManager.setState('playing');
window.gameDebug.resourceManager.loadTexture('/textures/brick.png');
```

## 🔧 Примеры использования

### Загрузка ресурсов через новый менеджер

```javascript
// Старый способ
const textureLoader = new THREE.TextureLoader();
textureLoader.load('/textures/brick.png', (texture) => {
    // использовать текстуру
});

// Новый способ
const texture = await game.resourceManager.loadTexture('/textures/brick.png');
// использовать текстуру
```

### Управление состоянием

```javascript
// Старый способ
game.isPaused = true;
game.isGameOver = false;

// Новый способ
game.stateManager.setState('paused');
game.stateManager.setState('playing');
```

### События

```javascript
// Старый способ
game.onMonsterSpawned(data);

// Новый способ
game.eventBus.on('monster_spawned', (data) => {
    // обработка события
});

// Эмитация события
game.eventBus.emit('key_collected', { keyId: 1 });
```

## 🚨 Отладка

### Проблемы и решения:

1. **Ресурсы не загружаются**
   ```javascript
   // Проверить статус загрузчика
   console.log(game.resourceManager.getLoadProgress());
   
   // Проверить ошибки
   game.eventBus.on('resource_error', (error) => {
       console.error('Resource error:', error);
   });
   ```

2. **События не работают**
   ```javascript
   // Проверить подписчиков
   console.log(game.eventBus.getAllEvents());
   
   // Включить debug mode
   game.eventBus.setDebugMode(true);
   ```

3. **Состояния не синхронизируются**
   ```javascript
   // Проверить текущее состояние
   console.log('Game state:', game.stateManager.getState());
   console.log('Possible transitions:', game.stateManager.getPossibleTransitions());
   ```
